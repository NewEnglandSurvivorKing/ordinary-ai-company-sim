# UI/UX 与视觉方向

> 本节收录关于界面基调、分阶段解锁、视觉美术方向的设计构想。

## UI 基调五步 + 风格方向

1. 确定主题；
2. UI 风格要符合主题；
3.  风格得迎合游戏性；
4. 在保证游戏性的前提下让玩家能够有效接受信息，并且能够消化信息，知道怎么操作；
5. 在保证接受信息的前提下注重可操作性，操作必须简单明了，符合一般经营游戏的直觉。

UI 做成现代企业风格即可；可以模仿真实的 ERP，或者直接用 ERP 界面。

找一点销量高的游戏做参考（瘟疫公司/群星/文明 6）

## 分阶段增加界面 + 大事件 CG/PPT + 教程文案开关

游戏分阶段的方式可以是通过增加界面或者说游戏功能，而不是粗暴的切换；

前期体现老板亲力亲为参与，后期让玩家决定决策和社会交互。

大事件直接切屏播静态图PPT cg。

## 游戏视觉 / 美术方向

游戏视觉问题：游戏相关视觉表现形式方向探讨，虽然本游戏重点不是视觉，但是良好的视觉呈现有助于吸引玩家。

做的美观点的工业控制台；毛子在这块有不少好玩的东西。

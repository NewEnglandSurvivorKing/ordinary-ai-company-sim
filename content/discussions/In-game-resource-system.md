# 游戏内资源系统

> 本节收录关于游戏内资源、转化方向的设计构想，如影响力、资金、人力等转化链条。


> "0xab16a9be"：我给AI的提示词就先把资源单位加上了（美元，TFops·h，KW），还有游戏形式选的是可暂停即时
>
> EDLSDPSY：这些太基础了，我说的细节不是这些。另外应该有必要区分汇率，主要是rmb对美元。
>
> "如更云如更"：一个想法，可以加入一个叫“影响力”的资源，通过消耗这个资源获得融资。影响力可以通过推出新模型，或者炒作自己的AI性能很强大，很危险来获得（当然后者也可能导致自己被政府封禁模型）
>
> "Sunbread OverClock"：影响力本身就是估值一环
>
> "如更云如更"：相当于一种，和电力，算力，影响力一样的显示资源，但到AGI时期加成就会骤降
>
> "某种有机智能"：玩家可以欺骗用户给模型降智，但是有用户流失风险、口碑风险、凝聚力下降(需要更高薪招人，或者工作效率下降)。

货币：待决定是否引入汇率系统。货币可通过融资、资金链运转获得。

人力：表现为人力点数，1点人力代表的人员数量未定。人力可通过招募、挖角、收购团队获得。

算力：表现为可用算力点数，影响训练的模型性能。可用算力提升有多种途径，如货币兑换（买卡/租卡），科技树加点（升级架构/自研芯片）。

影响力：消耗影响力点数获得融资，提高估值，引导舆论，甚至造成政治影响。影响力的积累可通过发布新模型、炒作来获得。

凝聚力：影响公司人员稳定性。暂未讨论如何获得。

电力：暂无具体影响描述。

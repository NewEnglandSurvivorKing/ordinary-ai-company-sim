# 游戏内资源系统

> 本节收录关于游戏内资源、转化方向的设计构想，如影响力、资金、人力等转化链条。

货币：待决定是否引入汇率系统。货币可通过融资、资金链运转获得。

人力：表现为人力点数，1点人力代表的人员数量未定。人力可通过招募、挖角、收购团队获得。

算力：表现为可用算力点数，影响训练的模型性能。可用算力提升有多种途径，如货币兑换（买卡/租卡），科技树加点（升级架构/自研芯片）。

影响力：消耗影响力点数获得融资，提高估值，引导舆论，甚至造成政治影响。影响力的积累可通过发布新模型、炒作来获得。

凝聚力：影响公司人员稳定性。暂未讨论如何获得。

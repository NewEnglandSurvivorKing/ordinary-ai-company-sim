# 开发控制台与协作工具

> 本节收录关于开发用控制台/作弊码和 issue 标签规范的设计构想。

## 控制台 / 作弊码

控制台用于游戏测试。

控制台功能：

- 无限电力
- 无限算力
- 无限计算卡
- 无限影响力
- 科技树全解锁
- 一键融资
- 一键调参数
- 一键国家补贴
- 一键造基建

**玩法变量：** `dev_cheat_console`（仅开发/测试用）

**系统反馈：** 控制台仅测试环境可用。
